Welcome to ADS Lib and Overleaf tutorial!
=========================================

*By: `Junjie Mao <https://sites.google.com/view/junjiemao-astroph>`*

This tutorial intends to assist authors in preparing tex files on
Overleaf with reference bibliography items managed via the ADS personal
library. Please see the two threads :ref:`sec:ads` and :ref:`sec:overleaf`
for more details.

Source code of the tutorial
===========================

built using Sphinx. Please use the latest version of Sphinx for optimal
performance. To build the manual from source, the command make html should
be enough to build the html pages in the build directory.
